# Sistema de Gestión de una Tienda de Videojuegos

## Descripción

Este proyecto fue desarrollado en Python con el objetivo de administrar el inventario y las ventas de una tienda de videojuegos.

El sistema permite:

* Agregar videojuegos al inventario.
* Mostrar todos los videojuegos registrados.
* Buscar videojuegos por código.
* Actualizar precios.
* Registrar ventas.
* Mostrar estadísticas del inventario.
* Eliminar videojuegos.
* Salir del sistema mediante un menú interactivo.

## Conceptos Aplicados

Durante el desarrollo del programa se utilizaron los siguientes conceptos de Python:

* Variables
* Condicionales (if, elif, else)
* Ciclos (while y for)
* Funciones
* Diccionarios anidados
* Listas
* Validación de datos
* Cálculos estadísticos básicos
